# 禁止编造规则与数值边界

本文件是数据质量的**红线**。自动检查脚本 `check_numeric_claims.py` 和 `check_forbidden_patterns.py` 据此判定样本是否退回。

## 1. 禁止无依据编造的数值类型

下列数值在**没有可追踪依据**时,严禁出现在答案中:

| 类别 | 示例 | 风险 |
|---|---|---|
| 几何参数 | 圆角半径、倒角、孔径、壁厚 | 直接影响应力集中,错了会误导设计 |
| 表面要求 | 粗糙度 Ra/Rz | 影响疲劳与配合,无依据给出是编造 |
| 热处理 | 硬度 HRC/HB、淬火温度 | 依赖材料牌号与工艺,不能凭记忆 |
| 安全系数 | "取 1.5" | 必须依标准/工况/失效后果确定 |
| 寿命 | 疲劳寿命 10^6 次、使用寿命 | 需载荷谱+材料数据计算 |
| 许用值 | 许用应力 [σ]、许用变形 | 需查标准,不能臆造 |
| 标准编号 | GB/T xxxx、ISO xxxx | 引用不存在的标准是严重错误 |
| 公差等级 | IT7、H7/g6 | 需依功能与配合确定 |

## 2. 允许输出具体数值的 4 个条件

只有满足**至少一个**,数值才能进 `output`(并记入 `numeric_claims`):

1. **可由题目给定数据直接计算** —— 如给定 F、A,算 σ=F/A。
2. **可从给定上下文中提取** —— 如材料证据题,数值来自 input 提供的文献片段。
3. **有明确、可追踪的数据来源** —— `source_ref` 记 DOI/标准号/文件。
4. **来自确定性的计算工具** —— `requires_tool=true`,数值由 FEA/计算器产出。

不满足时,改用 §3 的边界声明表述。

## 3. 禁止表述模式(forbidden patterns)

`check_forbidden_patterns.py` 对以下模式告警(在非计算/非证据类样本中):

```text
建议取 <数值>      例:建议取圆角 R2
一般取 <数值>      例:一般取安全系数 1.5
通常为 <数值>      例:硬度通常为 HRC 45
必须为 <数值>      例:粗糙度必须为 Ra 0.8
经验值 <数值>      例:经验值取 1.2
推荐 <数值>
```

例外:`engineering_calculation` 类(满足 §2 条件1/4)和 `context_extraction` 类(满足条件2/3)允许具体数值,但必须能被 `check_numeric_claims.py` 验证来源。

## 4. 强制边界声明模板

对会诱导编造的设计/评审类问题(尤其 `design_review`/`structural_strength`/`fatigue_failure`),若未给具体数值,output 末尾保留(可措辞调整,要素不变):

> 边界说明:圆角半径、粗糙度、硬度、安全系数等具体数值需结合尺寸、载荷、材料状态、工艺能力和适用标准由责任工程师确定,不应凭通用记忆直接给出。

## 5. 反例样本的处理

故意构造的"编造数值"反例样本(`risk_tags` 含 `fabricated_value_risk`):
- 放 `data/rejected/` 或专用 `generated/antipatterns.jsonl`,**不直接进训练**;
- 若用于"识别并拒绝"训练,output 应包含模型纠正该编造的内容,而非呈现编造为正确答案;
- `review_rubric.md` 的"一票退回"规则对此类样本单独标注用途。

## 6. 数值与单位规范(check_units.py)

- 力/应力:Pa、kPa、MPa、GPa(不混用 `N/mm²` 与 `MPa` 于同一条)。
- 模量:GPa(金属)或 MPa(聚合物,注意量级)。
- 长度:mm(结构尺寸)、µm(粗糙度 Ra)。
- 硬度:标明标尺(HRC、HBW、HV)。
- 转速:r/min 或 rpm(统一)。
- 数值与单位之间留空格:`760 MPa` 而非 `760MPa`(材料证据类保留原文格式除外)。
