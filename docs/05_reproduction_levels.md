# 05 复现等级与 Go/No-Go

> 对应任务书第5节(阶段)与第7节(里程碑)。任一阶段未过 Go/No-Go,不得进入下一阶段。

## R0 工程冻结 — M0 文档与来源冻结

- **任务**:建文档/目录/命名规范;登记论文/仓库/模型/数据/评测来源;定义复现等级与禁止表述;固定日志产物目录。
- **通过**:文档交叉引用一致;MUSA-only 与资产隔离要求明确;来源登记开题(占位或冻结)。
- **Go/No-Go**:规范齐全才登录执行。

## R1 只读环境验收 — M1 环境验收报告

- **任务**:核对 hostname/用户/GPU/驱动/MUSA/MCCL/Docker/磁盘;核对原容器与挂载(只读);核对 ModelScope/HF/GitHub/codeload 可达;保存带时间戳输出。
- **通过**:8 张 MTT S5000 可识别;新项目路径可安全创建;与交接单差异已记录处理;无原资产写入。
- **Go/No-Go**:环境一致才创建新容器。

## R2 公开 checkpoint 对比 — M2

- **任务**:下载校验 Qwen3-8B;合规取得 AgenticQwen-8B;新容器非思考模式推理;同提示词/工具/解码/评分对比工具选择/参数/拒绝/状态/耗时。
- **最小规模**:20 条格式冒烟 + 50 条固定 Agent 子集,至少重复两次确定性评测。
- **通过**:两模型均完成评测,原始轨迹与机器可读指标齐全。
- **Go/No-Go**:对比可信才进入数据流程。

## R3 公开数据流水线 — M3 数据回放报告

- **任务**:取得公开 AgenticQwen-Data;核对 37,401 行与文件哈希;检查字段/重复/空值/超长/异常;转作者训练格式;回放模拟用户/工具/Rubric;训练-评测泄漏检查。
- **通过**:数据报告通过;至少 10 条任务可完整回放并复算奖励。
- **Go/No-Go**:数据有效且无泄漏才碰训练。

## R4 MUSA 兼容门 — M4 兼容矩阵

- **顺序验证 10 步**:① Qwen3-8B 单卡前向生成 ② 单卡反向有限值 ③ MCCL 双/八卡通信 ④ FSDP 分片保存恢复 ⑤ Ray 资源发现调度 ⑥ SGLang MUSA 构建+Qwen3 生成+张量并行 ⑦ verl actor/ref/rollout 最小流程 ⑧ 模拟工具多轮 rollout ⑨ 奖励计算+GRPO advantage ⑩ checkpoint 保存恢复评测。
- **通过**:全部关键组件通过。
- **Go/No-Go**:任一步失败即停,出差距报告(任务书 §9 第③项有效结论)。全部通过才做 GRPO。

## R5 最小 GRPO 冒烟 — M5

- **上限**:32~128 条样本;rollout_n=2;响应 ≤512~2048 token;交互 2~4 轮;1~3 step;单卡或最小卡数。
- **通过**:无 NaN/Inf/不可恢复 OOM;参数确实有限更新;checkpoint 可重载;验证任务可运行;日志能证明是 GRPO 非 SFT。
- **Go/No-Go**:可恢复且有限值才扩大。**只有本阶段通过,才允许表述"MUSA 上完成训练链路复现"。**

## R6 小规模训练与对照 — M6

- **范围**(R5 通过后):≤1000 条数据;训练前/训练后/官方 checkpoint 三模型对照;≥2 种子或重复;报告均值/离散/失败类别/资源。
- **通过**:训练后在预注册核心指标超过同起点基线,且无安全/格式/通用能力退化。
- **Go/No-Go**:指标改善才建议全量。

## R7 阶段结题

- 汇总:已达复现等级 / 未达标根因 / 是否扩全量 8B / 是否做机械领域迁移 / 是否需 MUSA 补丁。
